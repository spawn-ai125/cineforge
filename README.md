# CineForge Core v1.0 🎬✨

An automated, hardware-friendly, open-source pipeline that instantly upgrades ordinary photos into cinematic masterpieces. **CineForge Core** simulates the raw organic sensor data of premium **Sony FX-series** cinema cameras, paired with hours of intricate color correction mimicking a seasoned professional working within **DaVinci Resolve**.

---

## 🚀 Key Features & Architectural Advancements

*   **Zero-Configuration (Drop & Run):** No complicated directory configuration or arguments required. Place the script directly into any folder containing pictures, launch it, and watch it generate cinematic copies instantly.
*   **Self-Supervised Neural Grading (CineNet v1 ):** Uses an on-the-fly, light-speed neural network optimization that trains itself on individual image characteristics in real-time, eliminating the need for massive multi-gigabyte datasets.
*   **True Optical Emulation Engines:**
    *   **Sony FX Cine-Alta Grain:** A luminance-mapped noise generator that integrates organic grain structures natively into the shadows and midtones while keeping highlights clean.
    *   **Pro Halation:** Simulates red light leaks and chemical color bleeding occurring near ultra-high contrast light boundaries.
    *   **Anamorphic Flare Streaks:** Projects directional blue-tinted optical horizontal lens leaks onto intense illumination vectors.
    *   **Bleach Bypass Processing:** Embeds high structural micro-contrasts inspired by classic Hollywood silver-retention film negatives.
*   **Hardware Friendly:** Optimized mathematical downsampling layers ensure that processing executes swiftly on standard computer CPUs without needing high-end desktop workstation GPUs.
